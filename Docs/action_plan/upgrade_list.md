 What to Buy (Shopping List)

  1. PSRAM chip - ESP-PSRAM64H or APS6404L (SOP-8, 3.3V)
  2. Breakout board (optional) - SOP-8 to DIP adapter if not soldering directly
  3. Decoupling capacitor - 100nF ceramic, place near VCC/GND


  Quick Checklist Before Buying

  ✓ QSPI interface (not SPI-only, not Octal-only)
  ✓ 3.3V operating voltage
  ✓ SOP-8 package (8 pins, 1.27mm pitch)
  ✓ 64Mbit / 8MB size
  ✓ Speed ≥ 84MHz
